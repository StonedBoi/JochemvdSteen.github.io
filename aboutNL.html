INTRODUCTIE

Ook al heb ik voornamelijk software development gestudeerd, het ding dat ik het meest geïnteresseerd in ben is het maken van games. Deze interesse begon toen een gamebedrijf naar mijn school kwam en een select groepje studenten les gaf over de kunst van het maken van games.

ERVARING

Tot nu toe heb ik met meerdere programma's gewerkt zoals Unity, Visual Studio, GitHub en Pycharm waarbij ik codeertalen als C#, Python, HTML/CSS/JavaScript en React heb gebruikt.

VAARDIGHEDEN

Ik het meest bekend met Unity en C# vanwege alle projecten waar ik aan heb gewerkt voor beide school en werk, en hoewel beperkt, heb ik nog steeds enig begrip van Python.

Helaas ben ik nog steeds onervaren op gebieden zoals SQL, HTML en vooral andere dingen die met web te maken hebben.

DOELEN

Het is mijn doel om in de toekomst een game developer te worden, maar ik ben nog steeds geïnteresseerd in andere vormen van programmeren.
Ik wil meer programmas leren zoals Unreal, en ook meer codeertalen zoals C++.
Maar boven alles, ik wil mensen entertainment geven.
