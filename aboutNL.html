INTRODUCTIE

Ik ben software development aan het doen, maar de grootste reden dat ik deze cursus volg is omdat ik een game developer wil worden. Deze interesse begon toen een game bedrijf een op bezoek bij mijn school kwam en ze ons lieten zien hoe de productie van een game in elkaar zit.

ERVARING

Tot nu toe heb ik met meerdere programma's gewerkt zoals Unity, Visual Studio, GitHub en Pycharm waarbij ik codeertalen als C#, Python, HTML/CSS/JavaScript en SQL heb gebruikt.

VAARDIGHEDEN

Tot nu toe ben ik het meest bekend met Unity en C# vanwege alle projecten waar ik aan heb gewerkt, maar ik ken ook wel wat over Python.

Helaas ben ik nog steeds zwak in gebieden zoals SQL, HTML en vooral andere dingen die worden gebruikt voor webpagina's en andere online databases. Dit is te zien in de Portfolio die u nu bekijkt.

DOELEN

Het is mijn doel om in de toekomst een game developer te worden, maakt niet uit hoe groot het bedrijf.
Ik wil meer programmas leren, bijvoorbeeld Unreal.
Maar boven alles, ik wil mensen entertainment geven.
