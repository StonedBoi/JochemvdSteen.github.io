Arcade Project

Dit was een groepsproject waarin ik samenwerkte met kunststudenten om zelf een game te maken. Onze beslissing was om een endless runner game te maken met een extra mechanic waarbij je doet tijdreizen en je je moet aanpassen aan de tijd zodat de onderlingen van de tijdgod hem niet naar jou stuurt.

Deze game is gemaakt in Unity in de loop van 8 weken en ik kan er niet meer trots op zijn, vooral omdat ik bijna alle mechanics heb gecodeerd

GameJam Project

De eerste keer dat ik aan een game heb gewerkt, was tijdens een GameJam-project waar het bedrijf 'Abstraction Games' developers naar onze school stuurde, en iedereen die geïnteresseerd was kon meedoen om de basics van game development te leren

Samen met hun hulp had ik geleerd Unity te gebruiken, en zij zijn ook de reden dat ik nu hier ben omdat dit project de reden was dat ik besloot een game developer te worden.

AR Target Practice

Dit is een eenvoudig AR-schietspel, het doel is om met je telefoon rond te kijken en spinnen te schieten. Er zijn twee versies: In één mode zullen de spinnen gewoon op foto's spawnen, op deze manier kun je een irl-configuratie hebben opzetten. In de tweede mode zal het spel een oppervlak detecteren en de spinnen erop spawnen.

Aangezien dit de eerste keer was dat ik aan een AR-project heb gewerkt, zullen er ongetwijfeld problemen zijn, maar in het algemeen ben ik blij met het resultaat en het is een grote verbetering voor de toekomst.
